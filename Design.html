Aniol64 design document

Overview

Aniol64 is an 8-bit, Z80-based microcomputer with built-in keyboard, display and serial port as well as extendable IO

Lead colour coding

Power supply: Blue
Ground: Black or Gray
Data: Red
Address: Green
Control: Yellow

Key rows: Red
Key coluns: Green
Special keys: Yellow

Memory

Memory map

0000-7FFF: ROM (28C256)
8000-FFFF: RAM (62C256 or DS1244Y)
8000-BFFF: general purpose RAM (AS6C4008)
C000-7FFF: bank-switchable NVRAM (AS6C4008)
